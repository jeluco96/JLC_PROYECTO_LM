Descripción:
Este proyecto consiste en el desarrollo de una página web para una tienda de juguetes antiguos.

Cambios realizados:

-->Carrusel de Imágenes: Se ha añadido un carrusel de imágenes para destacar algunos de los productos antiguos disponibles en la tienda.

-->Formulario de Inicio de Sesión: Se ha implementado un formulario de inicio de sesión con campos de usuario y contraseña. Se incluye una lógica de validación de usuario para verificar las credenciales ingresadas.

-->Check de Sesión Iniciada: Se ha agregado un check de sesión iniciada para confirmar visualmente si el usuario ha iniciado sesión correctamente.

-->Formulario Formspree: Se ha incluido un formulario de contacto que utiliza Formspree para enviar consultas por correo electrónico.

*Detalles a destacar:*

Los efectos CSS se han utilizado para agregar interactividad y dinamismo a la página, como animaciones al hacer hover sobre los elementos.
Se ha prestado especial atención a la usabilidad y la accesibilidad para garantizar una experiencia de usuario satisfactoria.
